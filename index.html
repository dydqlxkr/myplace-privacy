앱명: 나만의 지도 (MyMap)  |  패키지: io.launchings.mymap  |  개발자: 류용탁  |  최종 수정일: 2026년 6월 11일

1. 개요

나만의 지도(이하 "앱")는 광고 없이, 계정 가입 없이, 내 핀만 보이는 개인 지도 메모 서비스입니다.

앱은 로컬 퍼스트(local-first) 구조로 설계되어, 사용자의 핀 데이터·사진·메모 등 개인 기록은 기기 내 SQLite 데이터베이스에만 저장됩니다. 개발자 서버에 계정 정보나 개인 데이터를 저장하지 않습니다.

2. 수집·처리하는 정보

2-1. 기기에서만 사용하는 정보 (외부 전송 없음)

항목목적저장 위치
핀 데이터 (이름, 좌표, 메모, 태그, 주소, 카테고리 등) 지도 위 장소 기록 기기 내 SQLite DB
첨부 사진 핀에 연결된 이미지 표시 기기 내 앱 도큐먼트 디렉토리
현재 위치 (위도·경도) "내 위치" 버튼 동작 시 지도 중앙 이동 기기 메모리 (저장 안 됨)
앱 설정 (테마 등) 사용자 환경설정 유지 기기 내 SharedPreferences

현재 위치는 사용자가 "내 위치" 기능을 명시적으로 사용할 때만 일시적으로 읽히며, 앱 서버나 외부 서비스에 전송되지 않고, 기기 내에도 저장되지 않습니다.

첨부 사진은 갤러리에서 선택한 이미지를 앱의 비공개 저장 공간에 복사하여 저장합니다. 원본 갤러리 파일을 수정하거나 외부로 전송하지 않습니다.

2-2. 외부 서버로 전송되는 정보

앱은 장소(상호명) 검색 기능을 위해 다음 정보를 개발자가 운영하는 Cloudflare Worker 프록시를 통해 네이버 지역 검색 API로 전달합니다.

항목목적전송 대상서버 저장 여부
장소 검색어 (사용자 입력 키워드) 네이버 지역검색 API 호출 Cloudflare Worker 프록시 → 네이버 검색 API 저장 안 됨 (통과만)
기기 식별자 (UUID, 앱 설치 시 기기 내에서 생성) 일일 검색 한도 관리 (남용 방지) Cloudflare Worker KV 임시 카운터 (36시간 후 자동 만료)
  • 검색어와 기기 식별자는 검색 품질 분석, 마케팅, 타사 판매 등 다른 목적으로 사용하지 않습니다.
  • 기기 식별자(UUID)는 계정 식별이나 개인 신원 파악에 사용하지 않습니다.
  • 검색 결과 원본은 서버에 캐싱·저장하지 않습니다. 사용자가 선택한 장소의 이름·좌표만 핀으로 기기에 저장됩니다.

장소 검색 기능을 사용하지 않는 경우(롱프레스로 직접 핀 추가 등), 외부 전송이 발생하지 않습니다.

2-3. 네이버 지도 SDK

앱은 지도 표시를 위해 네이버 클라우드 플랫폼(NCP) Maps SDK를 사용합니다. SDK가 지도 타일 로드 및 서비스 운영을 위해 수집하는 데이터(기기 정보, IP 주소, 사용 통계 등)는 네이버 클라우드 플랫폼의 개인정보처리방침을 따릅니다.

네이버 클라우드 플랫폼 개인정보처리방침: https://www.ncloud.com/policy/privacy

3. 앱이 요청하는 권한

권한필요한 이유
인터넷 (INTERNET) 지도 타일 로드, 장소 검색 프록시 호출
정밀 위치 (ACCESS_FINE_LOCATION) "내 위치" 버튼으로 현재 위치를 지도에 표시
대략적 위치 (ACCESS_COARSE_LOCATION) 위치 권한의 보조 수단
미디어 이미지 읽기 (READ_MEDIA_IMAGES 등) 핀에 갤러리 사진 첨부 시

위치 권한과 사진 읽기 권한은 사용자가 해당 기능을 처음 사용할 때 명시적으로 요청하며, 거부해도 앱의 기본 기능(핀 추가·관리·지도 보기)은 정상 동작합니다.

4. 공유 기능

앱은 핀 정보를 다른 앱이나 사람에게 공유하는 기능을 제공합니다. 공유 시 전송되는 내용(핀 이름, 주소, 좌표, 네이버 지도 링크, 앱 간 전송 토큰)은 사용자가 공유 버튼을 누르는 시점에 OS 공유 시트를 통해 전달됩니다. 이 과정에서 개발자 서버에 데이터가 전송되지 않습니다.

단, "한 번에 받기 링크" 기능은 핀 데이터를 base64 인코딩하여 Cloudflare Worker 단축 URL 서비스를 통해 짧은 링크로 변환합니다. 이 경우 핀의 이름·좌표·주소 등이 링크 생성을 위해 프록시 서버를 경유합니다. 원본 데이터는 단축 링크의 키로만 일시 저장되며, 개인 식별이나 분석에 사용하지 않습니다.

5. 제3자 공유

개발자는 사용자 데이터를 광고주, 분석 서비스, 또는 제3자에게 판매하거나 공유하지 않습니다. 앱에는 광고 SDK, 분석 SDK, 소셜 로그인 SDK가 포함되어 있지 않습니다.

6. 데이터 삭제

앱의 모든 핀 데이터와 사진은 기기에 저장됩니다. 앱을 삭제하면 앱과 관련된 모든 데이터가 기기에서 함께 삭제됩니다. 개발자 서버에 저장된 개인 데이터가 없으므로 별도의 서버 삭제 요청이 필요하지 않습니다.

기기 식별자(UUID)는 Cloudflare Worker KV에 일일 검색 카운터로 36시간간 임시 보관됩니다. 이 카운터는 검색 횟수만 기록하며, 이름·연락처 등 개인 정보를 포함하지 않습니다. 삭제 요청은 아래 연락처로 문의 바랍니다.

7. 어린이 개인정보

앱은 만 13세 미만 아동을 대상으로 설계하지 않았으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 보안

앱 내 데이터는 기기의 앱 비공개 저장 공간에 보관됩니다. 외부 전송이 발생하는 경우(검색 프록시)에는 HTTPS를 사용합니다.

9. 개인정보처리방침 변경

방침이 변경될 경우 이 페이지를 통해 고지합니다. 중요한 변경이 있는 경우 앱 업데이트 노트에도 안내합니다.
